UTS SIGWeb

Esai mata kuliah SIG Web

Berisi tentang materi yang sudah dipelajari di mata kuliah Sistem Informasi Geografis berbasis web (Web-based Geographic Information System) di Departemen Teknik Geodesi, Fakultas Teknik, Universitas Gadjah Mada

Minggu 1, 2, 3

Untuk minggu pertama mempelajari Mengenai cara kerja internet dan istilah-istilah internet Mengenal komponen web: HTML, CSS dan JS Latihan membuat web sederhana dengan HTML dan CSS dan Membaca panduan praktik pembuatan web dengan WSL dan Linux. Pada minggu ini dipelajari peranan Javascript sebagai bahasa pemrograman klien (Client-side scripting) serta dasar-dasar AJAX dan perkembangan Web Modern

Minggu 4

Minggu 4 diisi dengan pembahasan mengenai Boostrap sebagai salah satu framework javascript yang paling banyak digunakan Praktek minggu ini, Membuat Halaman dengan Bootstrap V4 atau V5 Kustomisasi Bootstrap dengan Pingendo

Minggu 5 dan 6

Minggu ini membahas mengenai salah satu webMap API, yaitu LeafletJS. Praktikum pada minggu ini: Membuat halaman web sederhana pada LeafletJS, Layouting halaman web dengan Bootstrap, Buat layout peta Leaflet dengan Navbar Bootstrap Minggu 6 membahas mengenai LeafletJS, yaitu penambahan layer, pengenalan format GeoJSON dan instalasi Plugin. Praktikum minggu 6: - Tambahkan plugin ke dalam leaflet untuk memperkaya fungsi webgis. Plugin yang bisa ditambahkan (misalnya Leaflet Geolocation, Geocoding, Leaflet Graphic Scale, dll) dapat dilihat pada http://leafletjs.com/plugins.html - Gunakan QGIS atau GeoJSON.io untuk mengkonversi file Jogja.shp menjadi GeoJSON, kemudian tambahkan ke kerangka peta yang sudah dibuat. Gunakan panduan: https://leafletjs.com/examples/geojson/ - Mengkombinasikan bootstraps dengan leaflet yang dibuat (geojson + min 1 plugin) untuk membuat sebuah halaman web - Buat halaman web yang serupa dengan menggunakan QGIS2Web

Minggu 7

Pada modul ini akan dibahas mengenai bagaimana aplikasi Mergin Maps: QGIS in pocket sebagai perangkat berbasis mobile, QGIS sebagai perangkat desktop serta Mergin sebagai aplikasi GIS berbasis online dapat digunakan sebagai sarana untuk pengumpulan data dari lapangan secara kolaboratif. Pengumpulan data menggunakan kombinasi ketiga perangkat tersebut dapat digunakan dalam kegiatan survei lapangan, misalnya terkait dengan survei pelanggaran tata ruang, pemetaan aset, pemetaan tempat makan, pemetaan ATM maupun monitoring pembangunan. Pada Praktikum Minggu ke 7 ini, mahasiswa secara berkelompok yang terdiri dari maks 5 orang diminta untuk merancang sebuah kegiatan survei berupa Titik, Garis, Poligon dengan total seluruhnya minimal 30 fitur menggunakan kombinasi ketiga perangkat tersebut. Hasilnya dibuat dalam bentuk laporan dan dikumpulkan terakhir satu bulan dari sekarang.
